# 启动项目

## 依赖已安装完成 ✅

现在可以启动开发服务器了。

## 启动步骤

### 在命令提示符 (cmd) 中运行：

```bash
npm run dev
```

### 成功启动后，你会看到类似输出：

```
  ▲ Next.js 14.2.0
  - Local:        http://localhost:3000
  - Ready in 2.3s
```

## 查看网站

1. **打开浏览器**（Chrome、Edge、Firefox 等）
2. **访问地址**：`http://localhost:3000`
3. 你将看到鲲鹏航空的首页！

## 停止开发服务器

在 cmd 窗口中按 `Ctrl + C` 可以停止服务器。

## 下次使用

以后每次开发，只需要：
1. 打开 cmd（在项目文件夹中）
2. 运行 `npm run dev`
3. 在浏览器访问 `http://localhost:3000`

如果修改了代码，Next.js 会自动刷新页面显示最新内容。
