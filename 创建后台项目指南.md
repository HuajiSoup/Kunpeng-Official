# 新闻发布后台项目创建指南

## 方案一：在同一工作空间创建后台项目（推荐）

### 步骤 1：创建后台项目文件夹

在 Cursor 中，你可以：

1. **使用终端创建文件夹**：
   ```powershell
   cd f:\官网test
   mkdir admin
   cd admin
   ```

2. **或者直接在 Cursor 文件资源管理器中右键创建新文件夹**

### 步骤 2：初始化 Next.js 项目

在 `admin` 文件夹中运行：

```powershell
npx create-next-app@latest . --typescript --tailwind --app --no-src-dir --import-alias "@/*"
```

或者使用交互式创建（推荐）：
```powershell
npx create-next-app@latest admin-panel
```

### 步骤 3：安装后台管理所需依赖

```powershell
cd admin
npm install
npm install @tanstack/react-query axios react-hook-form @hookform/resolvers zod
npm install date-fns react-hot-toast
npm install lucide-react  # 如果还没有安装
```

### 步骤 4：项目结构建议

```
admin/
├── app/
│   ├── layout.tsx          # 后台布局（包含侧边栏）
│   ├── page.tsx            # 后台首页/仪表盘
│   ├── login/
│   │   └── page.tsx        # 登录页面
│   ├── news/
│   │   ├── page.tsx        # 新闻列表
│   │   ├── create/
│   │   │   └── page.tsx    # 创建新闻
│   │   └── [id]/
│   │       └── page.tsx    # 编辑新闻
│   └── api/
│       └── news/
│           └── route.ts    # 新闻 API 路由
├── components/
│   ├── admin/
│   │   ├── Sidebar.tsx     # 侧边栏导航
│   │   ├── Header.tsx       # 后台头部
│   │   ├── NewsForm.tsx    # 新闻表单组件
│   │   └── NewsTable.tsx   # 新闻列表表格
│   └── ui/                 # 通用 UI 组件
├── lib/
│   ├── api.ts              # API 客户端
│   ├── auth.ts             # 认证相关
│   └── db.ts               # 数据库连接（如果使用）
└── types/
    └── news.ts             # 新闻类型定义
```

## 方案二：创建完全独立的新项目

### 步骤 1：在 Cursor 中打开新文件夹

1. 点击 Cursor 菜单：`File` → `Open Folder...`
2. 选择或创建新文件夹，例如：`f:\新闻后台管理系统`
3. 点击"选择文件夹"

### 步骤 2：初始化项目

在新文件夹中打开终端（Terminal → New Terminal），运行：

```powershell
npx create-next-app@latest . --typescript --tailwind --app --no-src-dir --import-alias "@/*"
```

### 步骤 3：安装依赖

```powershell
npm install
npm install @tanstack/react-query axios react-hook-form @hookform/resolvers zod
npm install date-fns react-hot-toast lucide-react
```

## 推荐技术栈

### 核心框架
- **Next.js 14+** (App Router) - 与官网保持一致
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架

### 状态管理和数据获取
- **@tanstack/react-query** - 数据获取和缓存
- **axios** - HTTP 客户端

### 表单处理
- **react-hook-form** - 表单管理
- **zod** - 数据验证
- **@hookform/resolvers** - 表单验证解析器

### UI 增强
- **react-hot-toast** - 消息提示
- **date-fns** - 日期处理
- **lucide-react** - 图标库

### 可选：数据库和认证
- **Prisma** + **SQLite/PostgreSQL** - 数据库 ORM
- **NextAuth.js** - 身份认证
- **bcryptjs** - 密码加密
