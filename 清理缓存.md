# 清理 Next.js 缓存

## 在 PowerShell 中运行以下命令：

### 方法 1：使用 Remove-Item（推荐）
```powershell
Remove-Item -Recurse -Force .next
```

### 方法 2：使用 rm 别名
```powershell
rm -r -Force .next
```

### 方法 3：如果目录不存在也不报错
```powershell
if (Test-Path .next) { Remove-Item -Recurse -Force .next }
```

## 在 CMD（命令提示符）中运行：

### 使用 rmdir 命令
```cmd
rmdir /s /q .next
```

## 完整步骤：

1. **停止开发服务器**：按 `Ctrl + C`

2. **清理缓存**（PowerShell）：
   ```powershell
   Remove-Item -Recurse -Force .next
   ```

3. **重新启动**：
   ```powershell
   npm run dev
   ```

4. **等待编译完成**，然后访问 `http://localhost:3000/testing`
